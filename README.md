# FoodExpressDB

## 📌 Project Overview

FoodExpressDB is a MySQL-based Food Delivery Management System designed to manage customers, restaurants, food items, orders, payments, deliveries, and reviews. The database is designed using normalization principles and includes business-oriented SQL objects.

## 🚀 Features

- Customer Management
- Restaurant Management
- Food Menu Management
- Order Processing
- Payment Management
- Delivery Tracking
- Customer Reviews
- Views for Reporting
- Stored Procedures
- SQL Functions
- Business Reports

## 🛠 Technologies Used

- MySQL 8.x

- SQL
- Git
- GitHub

## ▶️ How to Run

1. Open MySQL Workbench.
2. Create a new schema (optional).
3. Open `SQL/FoodExpressDB.sql`.
4. Execute the script.
5. Verify that the tables and sample data are created.
