# 🍔 FoodExpressDB – Enterprise Food Delivery Management System

## 📖 Project Overview

**FoodExpressDB** is an enterprise-level MySQL database project that simulates a real-world online food delivery platform such as Swiggy or Zomato. It demonstrates database design, normalization, SQL programming, and business reporting.

## 🚀 Features

- Customer Management
- Restaurant Management
- Food Menu Management
- Order Management
- Payment Management
- Delivery Tracking
- Coupon Management
- Customer Reviews
- Audit Logging
- Business Reports

## ▶️ How to Run

1. Open MySQL Workbench.
2. Execute SQL scripts in numerical order:
   - 01_Create_Database.sql
   - 02_Create_Tables.sql
   - 03_Insert_Master_Data.sql
   - 04_Insert_Transaction_Data.sql
   - Continue through all remaining scripts.
3. Verify data using SELECT queries.
4. Execute views, procedures, functions, triggers and reports.
